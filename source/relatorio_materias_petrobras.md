# Relatório Detalhado de Matérias por Profissão - Petrobras

Este relatório apresenta um detalhamento exaustivo das matérias por profissão, segmentado por nível de escolaridade (Médio/Técnico e Superior), com base nos editais da Petrobras de 2023.2 e 2021. O objetivo é fornecer uma base sólida e precisa para a criação de conteúdo para o seu SAAS, simulando o cenário para o edital de 2026.

---

## 1. Nível Médio / Técnico (Base: Edital Petrobras 2023.2)

Os cargos de Nível Médio/Técnico no edital 2023.2 exigem formação técnica profissionalizante. As provas para todas as ênfases tiveram duração de 4 horas.

## 3. Recomendações para o SAAS (Edital 2026)

Para otimizar a experiência do usuário em seu SAAS, sugere-se a seguinte estrutura:

*   **Módulos de Conhecimentos Básicos:** Crie módulos separados para Língua Portuguesa, Matemática e Língua Inglesa, que podem ser atribuídos automaticamente com base no nível de escolaridade selecionado pelo usuário.
*   **Módulos de Conhecimentos Específicos:** Cada profissão deve ter um módulo dedicado, contendo os tópicos detalhados em seus respectivos blocos. Isso permitirá que o usuário selecione sua área de interesse e acesse o conteúdo programático específico de forma organizada.
*   **Flexibilidade:** Permita que o usuário personalize seu plano de estudos, selecionando ou desmarcando tópicos específicos dentro de cada matéria, conforme sua necessidade.
*   **Atualizações:** Mantenha um sistema de fácil atualização para incorporar eventuais mudanças nos próximos editais da Petrobras, especialmente para o edital de 2026.

> **Aviso:** Este relatório foi compilado com base nos editais mais recentes disponíveis. É fundamental que, ao se aproximar o edital de 2026, seja feita uma nova verificação para confirmar possíveis alterações no conteúdo programático ou na estrutura dos cargos. No entanto, a base apresentada aqui é robusta e historicamente consistente com os padrões da Petrobras. [1] [2] [3]
