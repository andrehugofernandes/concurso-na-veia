💻 COMO USAR (SUPER RÁPIDO)
════════════════════════════════════════════════════════════════════════════

Ao criar uma aula nova com 10 módulos:

  1. Import:
     import { getModuleVariant, getModuleColor } from "@/lib/moduleColors";

  2. ModuleBanner (auto-usa cor do módulo):
     <ModuleBanner index={1} titulo="..." />

  3. ModuleSectionHeader (passa variant):
     <ModuleSectionHeader index={1} variant={getModuleVariant(1)} title="..." />

  4. CardCarousel (passa cor):
     <CardCarousel corIndicador={`bg-${getModuleColor(1)}`} ... />

FIM! Repita para módulos 2-10 com index={2-10}.
