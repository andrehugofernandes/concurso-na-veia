# Sala de Guerra: Padronização das Aulas (43 Arquivos)

## Diretrizes de Padronização
- [ ] **1. Indexação dos cards:** Sequência correta dentro de cada módulo.
- [ ] **2. Sistema de Cores:** Banner inicial na mesma cor que o header dos cards numerados.
- [ ] **3. Resumo do Módulo:** Apenas um por módulo, com 4 abas (vídeo, resumo visual, macete, áudio), posicionado imediatamente antes do quiz.
- [ ] **4. Quiz:** Indexação = quantidade de cards (último card do módulo) e *não* exibindo o número do módulo no cabeçalho.
- [ ] **5. Limite de Linhas:** Garantir arquivos com > 2500 linhas. Todo o conteúdo dos quizzes e textos hiper densos devem permanecer intrínsecos ao arquivo.
- [ ] **6. Acordões:** Utilizar o padrão Acordeon-Carrossel para conteúdos espremidos.
- [ ] **7. Preservação Total:** Nenhuma perda de conteúdo textual ou didático.
- [ ] **8. Preservar StickyModuleNav:** Não alterar este componente.
